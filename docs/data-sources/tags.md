# komodo_tags (Data Source)

Lists all Komodo tags visible to the authenticated user.

## Example Usage

```terraform
data "komodo_tags" "all" {}

output "tag_names" {
  value = [for t in data.komodo_tags.all.tags : t.name]
}
```

## Schema

### Read-Only

- `tags` (List of Object) The list of tags. (see [below for nested schema](#nestedatt--tags))

<a id="nestedatt--tags"></a>
### Nested Schema for `tags`

Read-Only:

- `id` (String) The tag identifier (ObjectId).
- `name` (String) The tag name.
- `color` (String) The tag color.
- `owner` (String) The owner id (auto-generated by Komodo API).
